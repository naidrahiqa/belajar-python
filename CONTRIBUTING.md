### 1. Report Bugs
Jika menemukan bug atau error:
- Buka [GitHub Issues](https://github.com/username/belajar-python/issues)
- Jelaskan bug dengan detail
- Sertakan screenshot jika memungkinkan
- Jelaskan langkah untuk reproduce bug

### 2. Suggest Features
Punya ide untuk improvement?
- Buka [GitHub Issues](https://github.com/username/belajar-python/issues)
- Gunakan label "enhancement"
- Jelaskan feature yang diinginkan
- Jelaskan mengapa feature tersebut berguna

### 3. Submit Pull Request

#### Setup Development Environment
```bash
# Fork repository
# Clone fork Anda
git clone https://github.com/YOUR_USERNAME/belajar-python.git
cd belajar-python

# Buat branch baru
git checkout -b feature/your-feature-name
```

#### Membuat Perubahan
1. **Untuk Module Baru**:
   - Buat folder di `modules/XX-nama-module/`
   - Buat `README.md` dengan format yang sama
   - Tambahkan `examples/` dan `exercises/`

2. **Untuk Perbaikan**:
   - Edit file yang relevan
   - Pastikan formatting konsisten
   - Test semua contoh kode

3. **Untuk Examples/Exercises**:
   - Tambahkan komentar yang jelas
   - Test kode untuk memastikan berjalan
   - Sesuaikan dengan level module

#### Submit PR
```bash
# Commit changes
git add .
git commit -m "feat: Add [deskripsi singkat]"

# Push to your fork
git push origin feature/your-feature-name

# Buat Pull Request di GitHub
```

## ğŸ“ Guidelines

### Commit Messages
Gunakan format:
- `feat: Add new module XX`
- `fix: Fix typo in module YY`
- `docs: Update README`
- `style: Format code in module ZZ`

### Code Style
- Gunakan **4 spasi** untuk indentasi
- Follow **PEP 8** untuk Python code
- Komentar dalam **Bahasa Indonesia**
- Nama variabel yang **descriptive**

### Documentation
- Penjelasan dalam **Bahasa Indonesia**
- Gunakan **contoh yang jelas**
- Sertakan **komentar** di code
- Format markdown yang **konsisten**
